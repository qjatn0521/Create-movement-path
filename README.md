# Create Movement Path (Grid Pathfinding & Smooth Movement)
광운대학교 가상세계 3차 과제 (개인 프로젝트)

# 📌 프로젝트 개요
본 프로젝트는 Grid 기반 맵 환경에서 캐릭터의 이동 경로를 생성하고, 이를 부드럽게 따라 이동하도록 구현한 프로젝트입니다.
Unity 엔진에서 제공하는 NavMesh 기능을 사용하지 않고, 그래프 구성, A* 경로 탐색, 이동 보간 알고리즘을 직접 구현하는 것을 목표로 하였습니다.

단순히 최단 경로를 찾는 데 그치지 않고, 실제 게임 환경에서 자연스러운 이동을 표현하기 위해
Bezier 곡선 기반의 이동 경로 보간, 이동 방향 회전 처리, 애니메이션 연동까지 포함하여
경로 생성부터 캐릭터 이동 제어 전 과정을 코드 레벨에서 구현하였습니다.
